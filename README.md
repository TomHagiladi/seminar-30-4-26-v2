# Authentic Learning & Reassessment in the AI Era

Workshop materials for Tom Hagiladi's 2-hour Zoom session on **30.4.2026** with the Kibbutzim College "Advanced Community" #4. The workshop explores authentic learning, heutagogy, and assessment in the AI era.

## What's here

- `index.html` — the 19-slide deck (RTL Hebrew). What Tom screen-shares.
- `app/index.html` — the companion SPA participants open on their phones. Hash routing.
- `app-qr.png` — QR code pointing to the companion app.

Everything is a static file. No build, no server, no API keys.
